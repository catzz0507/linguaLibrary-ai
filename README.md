# LinguaLibrary AI

A Gemma 4-powered language learning RPG platform, starting with JLPT Japanese.

## Overview

LinguaLibrary AI transforms language learning into an RPG battle experience.  
The current prototype focuses on JLPT Japanese learning as the first playable module.

## Key Features

- JLPT-style Japanese quiz gameplay
- RPG battle-based learning loop
- AI-generated question pipeline
- Structured quiz data format
- Unity-based prototype
- Designed for future multilingual expansion

## Long-Term Vision

JLPT is the first module, not the final destination.

The system is intended to expand into a unified language learning platform supporting:

- Korean TOPIK
- English TOEIC / IELTS
- Chinese HSK
- Spanish DELE
- French DELF
- German Goethe-Zertifikat

## Tech Stack

- Unity
- C#
- Gemma 4
- Local LLM API workflow
